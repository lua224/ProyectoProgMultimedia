Esta aplicación tiene el objetivo de poder llevar un control de una colección personal. 
La he hecho de muñecas porque es lo que yo colecciono, y ninguna de las aplicaciones o 
páginas que he encontrado me ha gustado lo suficiente como para continuar usándola. 

De momento tiene una lista con nombres de muñecas y una imagen genérica que acompaña 
cada nombre. Hay un botón para marcar si esa muñeca forma parte de la colección o no.
